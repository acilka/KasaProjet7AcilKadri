# Kasa — Plateforme de location (React)

Refonte front-end d’une plateforme de location immobilière réalisée avec React à partir de maquettes Figma.

## Objectifs pédagogiques

* Création d’une SPA React
* Routing dynamique
* Composants réutilisables
* Gestion des données locales

## Technologies utilisées

* React
* React Router
* SCSS / CSS Modules
* Vite

## Fonctionnalités

* Pages dynamiques logements
* Slideshow d’images
* Collapse animés
* Gestion erreurs 404
* Navigation SPA fluide

## Compétences démontrées

* Architecture composants scalable
* Props & state management
* Routing avancé
* Structuration projet professionnelle

## Installation

```bash
git clone https://github.com/acilka/KasaProjet7AcilKadri
```
